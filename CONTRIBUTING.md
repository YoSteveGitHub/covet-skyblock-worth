# Contributing to the CovetSkyblock Economy

Thank you for helping improve the CovetSkyblock economy.

To maintain fairness and stability, all contributions must follow the rules below.

---

## ‚úÖ Before Submitting a Pull Request

Ensure:

- All YAML formatting is valid
- No added comments
- Only the intended item(s) are modified
- You provide a reason for all changes

Please limit changes to only one or a few items per pull request.

---

## üìã Required Format

Your pull request must follow this format:

```
Item: <>
Current Buy: <>
Current Sell: <>
Proposed Buy: <>
Proposed Sell: <>
Reason For Change: <>   
```

For example:
```
Item: iron_ingot
Current Buy: 380.00
Current Sell: 160.00
Proposed Buy: 90.00
Proposed Sell: 40.00
Reason For Change: Iron farms produce very high output, causing inflation. Reducing the value helps stabilize the market.
```

---

## ‚ùå What Will Be Rejected

- Changes without justification
- Emotional or opinion-based arguments
- Large sweeping rebalance pull requests without prior discussion
- Duplicate proposals
